# Recipe Manager

## Описание
Проект представляет собой систему управления рецептами. Позволяет создавать рецепты, добавлять ингредиенты, масштабировать порции и формировать список покупок.

### Установка
```bash
git clone https://github.com/Rastok314/HomeWork2.git
cd <корневая папка проекта>
pip install -r requirements.txt
```
### Запуск Тестов:
```bash
pytest
```
